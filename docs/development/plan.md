# AiKv 开发计划

> **最后更新**: 2026-01-16
> **状态**: 文档模板，待填充

## 项目概述

AiKv 是一个基于 AiDb v0.6.3 的高性能 Redis 协议兼容层实现。

## 当前版本

**版本**: v0.1.0
**状态**: 开发中

## 已完成功能

### 核心功能 ✅
- Redis 协议兼容层 (RESP2/RESP3)
- 基础数据结构 (String, List, Hash, Set, ZSet)
- Lua 脚本支持
- 集群基础功能 (MetaRaft)

### 存储层 ✅
- MemoryAdapter (内存存储)
- AiDbAdapter (持久化存储，支持所有数据类型)
- 存储层架构重构完成

### 可观测性 ✅
- 结构化日志
- Prometheus 指标
- 慢查询日志

## 开发中功能

### 集群功能 🚧
- 在线槽位迁移
- 集群故障转移
- 节点自动发现
